What is Markup Language ?

A computer language that consists of easily understood keywords, names, or tags that help format the overall view of a page and the data it contains

Some Markup Language :-

2. XHTML

HTML Tag :-

Anything between the Angular Bracket like - p , a, button, div, img, etc

Tags can be self closing or (Opening and Closing tag)

HTML Element :-

The html element is to be from Opening tag to closing tag is known as HTML element Like Paragraph, Body, HTML, div, division etc..

Syntax :-

Opening Tag + Content + Closing Tag

Attributes :-

Attributes provides additional functionality/information , it is always written in the Opening tag like - < img src=".." alt=".."> here src and alt is a Attributes

Anchor Element :-

The Anchor, HTML element with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Content within each should indicate the link's destination.
